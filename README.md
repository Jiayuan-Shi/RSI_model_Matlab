Usage
=  
Run command:
self(a,b,c) 

Explanation: 
-
a (Number of old haplotypes)  
b (Number of new haplotypes)  
c (Number of iterations)  

Return
=
Column: Haplotype frequency for a single iteration  
Row: A genotype frequency iteration from 1 to c generations
